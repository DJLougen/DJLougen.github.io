01 Research

Methods & Architecture

λ ∿ Σ

SABER

Spectral Analysis-Based Entanglement Resolution

CCA-based surgical refusal editing in post-trained models. Decomposes refusal behaviors into spectral components, enabling precise editing without collateral capability loss. Applied across Qwen, Ornstein, and Harmonic model families.

GitHub →
∇ ⊕ θ

NSC-ACE

GRPO + Latent Steering Vectors

Neural Steering via Controlled Activation — Contextual Embedding. Combines Group Relative Policy Optimization with latent steering vectors for improved instruction-following and controllability in open-weight models.

GitHub →
Δ ◬ ∞

BOREAL

From-Scratch Pretraining · DeltaNet Hybrid

Building foundation models from the ground up with DeltaNet hybrid architecture. Scaling path: 250M → 2B → 10B-MoE. Focused on efficient attention mechanisms and novel training curricula for open-weight pretraining.

GitHub →
