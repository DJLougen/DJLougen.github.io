// Research

Methods & Architecture

SABER

Spectral Analysis-Based Entanglement Resolution

CCA-based surgical refusal editing in post-trained models. Decomposes refusal behaviors into spectral components, enabling precise editing without collateral capability loss. Applied across Qwen, Ornstein, and Harmonic model families.

GitHub →

NSC-ACE

GRPO + Latent Steering Vectors

Neural Steering via Controlled Activation — Contextual Embedding. Combines Group Relative Policy Optimization with latent steering vectors for improved instruction-following and controllability in open-weight models.

GitHub →

BOREAL

From-Scratch Pretraining · DeltaNet Hybrid

Building foundation models from the ground up with DeltaNet hybrid architecture. Scaling path: 250M → 2B → 10B-MoE. Focused on efficient attention mechanisms and novel training curricula for open-weight pretraining.

GitHub →

// Product

SelfTrace

SelfTrace

Proprietary

Full-stack AI agent observability & introspection

SelfTrace is a commercial tracing and introspection platform for AI agent systems. It provides deep visibility into agent decision-making, tool-call chains, and state transitions — enabling teams to debug, audit, and optimize autonomous agent behavior in production.

Agent Trace Visualization

Full execution traces with tool calls, LLM generations, and branching logic rendered as interactive timelines.

State Introspection

Inspect agent memory, context windows, and intermediate representations at any point in the execution graph.

Performance Profiling

Latency breakdown, token cost attribution, and bottleneck detection across multi-step agent workflows.

Production Monitoring

Real-time dashboards and alerting for deployed agent systems. Anomaly detection on trace patterns.
